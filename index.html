HOT n Hazy – Momo Ordering Web App

HOT n Hazy is a dynamic full-stack online ordering platform for a street-style momo joint in Bokaro Steel City. Interactive features include animated cursors, dynamic testimonials, and real-time menu fetching. The backend efficiently manages orders with database integration and auto-email confirmations for both admin and customer, bringing professionalism to a small business setup.

Backend: PHP (server-side logic, email automation)
Database: MySQL (orders, momos)
Tools: Figma, XAMPP, PHPMyAdmin
